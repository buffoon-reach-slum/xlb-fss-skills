# xlb-fss-skills

FSS/XLB 团队 Codex skills 集合，用于沉淀接口字段、XlbFetch、列表页、弹窗和代码 review 等项目规范。

## CC Switch 导入

在 CC Switch 的 Skills 页面打开 Repository Management，点击 Add Repository，按 GitHub 仓库信息填写：

- Owner: GitHub 用户名或组织名
- Name: xlb-fss-skills
- Branch: main
- Subdirectory: skills

仓库结构保持为：

```text
skills/
  skill-name/
    SKILL.md
```

每个 skill 目录需要包含 `SKILL.md`，复杂资料可放到该 skill 自己的 `references/`、`scripts/` 或 `assets/` 下。
